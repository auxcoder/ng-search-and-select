Component bindings
  • ng-model: will be bound to controller
  • options: collection of the select element
  • id-key string: property of each of the select element
  • label-from-keys string: space delimited collections of keys used to build a label for the select element
  • placeholder-text string: will be used when no ng-model by default as the select placeholder
  • disabled boolean: will controll if the select input is enabled/disabled
  • select-icon string: an icon class name used for the select element 

<search-and-select
	class="search-select"
	ng-model="lesson"
	options="options"
	id-key="id"
	label-from-keys="name"
	placeholder-text="Select System Type"
	select-icon="glyphicon glyphicon-chevron-down"
	ng-change="onChangeSelectSystem(lesson)"
	disabled="isDisabled"
>
</search-and-select>

Use

The element can be disabled setting the binding key disabled to false, added to control cascade selects.

ng-module: {{ lesson | json }}
